# Contributing

Merci de votre intérêt pour contribuer !

- Ouvrez une issue pour discuter d'une fonctionnalité ou d'un bug.
- Fork puis créez une branche pour votre travail (`git checkout -b feature/ma-fonction`).
- Respectez le style existant (HTML/CSS/JS). Utilisez `deno fmt` si vous avez Deno.
- Ajoutez des commits clairs et une description utile dans la PR.

Nous apprécions les petites améliorations et corrections de bugs.
